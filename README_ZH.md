# Dev Guideline

使用 Claude Code 构建应用的开发规格模板。多语言支持 (KO/EN/JA/ZH)。

## 模板

| 模板 | 说明 | 语言 |
|----------|-------------|-----------|
| Detailed | 包含 API、数据模型、界面、阶段的完整规格 | KO/EN/JA/ZH |
| Brief | 简洁的单页规格 | KO/EN/JA/ZH |
| Ultra-Simple | 面向非程序员，只需填空 | KO/EN/JA/ZH |
| Template-Based | 适用于使用 `_template_latest` 的项目 | KO/EN/JA/ZH |

## 使用方法

1. 根据自身经验水平选择模板
2. 复制并填写空白处
3. 将完成的规格交给 Claude Code
4. Claude Code 逐步实现
