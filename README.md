# Dev Guideline

Development specification templates for building apps with Claude Code. Multi-language support (KO/EN/JA/ZH).

## Templates

| Template | Description | Languages |
|----------|-------------|-----------|
| Detailed | Full spec with API, data model, screens, phases | KO/EN/JA/ZH |
| Brief | Compact single-page spec | KO/EN/JA/ZH |
| Ultra-Simple | For non-programmers, just fill in blanks | KO/EN/JA/ZH |
| Template-Based | For projects using `_template_latest` | KO/EN/JA/ZH |

## Usage

1. Pick a template matching your experience level
2. Copy and fill in the blanks
3. Give the completed spec to Claude Code
4. Claude Code implements step by step
