# Dev Guideline

Claude Code로 앱을 빌드하기 위한 개발 명세서 템플릿. 다국어 지원 (KO/EN/JA/ZH).

## 템플릿

| 템플릿 | 설명 | 언어 |
|----------|-------------|-----------|
| Detailed | API, 데이터 모델, 화면, 단계를 포함한 전체 명세서 | KO/EN/JA/ZH |
| Brief | 한 페이지로 된 간결한 명세서 | KO/EN/JA/ZH |
| Ultra-Simple | 비개발자용, 빈칸만 채우면 완성 | KO/EN/JA/ZH |
| Template-Based | `_template_latest` 사용 프로젝트용 | KO/EN/JA/ZH |

## 사용법

1. 본인의 경험 수준에 맞는 템플릿 선택
2. 복사 후 빈칸 채우기
3. 완성된 명세서를 Claude Code에 전달
4. Claude Code가 단계별로 구현

## 추가 포함 내용

- `프로젝트_기본절차_및_셋업_가이드.md` — 프로젝트 전체 셋업 가이드
- `개발상세서_template_latest_packagelist.md` — PackageList 개발 명세서 (예시)
